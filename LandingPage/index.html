HOW TO SETUP THIS PROJECT?

Required Software:

  • download node.js
  • download Unity 3D

Run Project:

  1. Run the Server.js file inside ludoNodeServer using "node server.js" cmd.
  2. Import the unity project in unity.
  3. Open the Game scene.
  4. hit run.

Node Modules:

  • express
  • socket.io
  • shortid

NOTE: Pass and play option is not available you can either play it online by hosting the node server project on a server of your choice like heroku, aws, azur, google cloud. You can also play it on lan.

NOTE: if you have uploaded the node.js project to a server e.g. Heroku. Select "Socket" game object in the unity hierarchy and uncheck "isTesting" and chenge the "baseUrl" to your domain name replacing "https" with "wss" or "http" with "ws".
